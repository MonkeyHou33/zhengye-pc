# zhengye-pc

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
### 功能实现
- 头部导航栏左侧导航栏均可点击与面包屑交互回显

## 1.协议主体
- 协议客户通过mock模拟接口数据点击后可以弹窗，有确认校验
- 协议主体按钮均可点击回显
- 购进渠道下拉框点击选择后通过mock模拟接口，获取数据，table显示渠道数据

## 2.产品政策
- 单选、多选下拉框均可点击回显

## 3.补充协议
- 新增按钮可以点击，点击后新增补充协议
- 删除按钮可以点击，点击后删除当前选中补充协议

## 保存
- 做了协议客户必须选择项校验

### 使用技术
- vue 2.6.14
- element-ui 2.15.12
- less 4.1.3
- axios 1.3.2
- vuex 3.6.2
- vue-router 3.6.5
- mock.js 1.1.0
